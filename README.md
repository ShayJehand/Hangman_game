Hangman Game 🎮

A classic Hangman word guessing game built with Python. Guess the hidden word one letter at a time before the hangman is fully drawn!


How to Run

Make sure you have Python installed, then run:

bashpython hangman.py


How to Play


A random word is chosen secretly
You are shown blank spaces representing each letter of the word
Guess one letter at a time by typing it and pressing Enter
If the letter is in the word, it gets revealed in the correct position
If the letter is wrong, a body part is added to the hangman figure
You have 6 wrong guesses before the hangman is complete and you lose
Guess all the letters correctly before that to win!
